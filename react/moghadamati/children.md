# 📦 مفهوم `children` در React

## 🧠 `children` چیست؟
در React، `children` یک **prop ویژه** است که به‌صورت خودکار توسط React به هر کامپوننت ارسال می‌شود و شامل **محتوایی است که بین تگ باز و بسته‌ی کامپوننت قرار می‌گیرد**.

  سلام دنیا
</MyComponent>
